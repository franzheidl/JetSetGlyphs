JetSetGlyphs Help

Import Glyphset From Font

Will import the glyphset from the current font.

Import Glyphset From File

Will import a glyphset from a space-separated .txt-file.

Edit Glyphset In-Flight

To remove indivdual glyphs from your passenger list, click them and hit backspace/delete. Hold Shift to select multiple glyphs at once for deletion.

Create Glyphs In Font

Will create the glyphs from your imported/edited glyphset in your current font. Glyphs that already exist in your current font will be respected, only glyphs that are not present in your current font will be created as template glyphs.

Creating Glyphs can not be undone.

Export Glyphset to File

Will create a space-separated .txt-file of your imported/edited glyphset.

Copy Glyphset to Clipboard

Will copy a space-separated string of your glyphset to the clipboard, ideal for pasting into a new default character set at RoboFont > Preferences > Character Set.

Update JetSetGlyphs

Use Mechanic or pull from https://github.com/franzheidl/jetsetglyphs.
